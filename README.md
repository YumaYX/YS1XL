# YS1XL

Reusable procedures and functions for Excel automation.

## how to use

```sh
curl -o module.bas.txt https://raw.githubusercontent.com/YumaYX/YS1XL/refs/heads/main/module.bas
```

## for development

### split

```sh
cargo run -- split module.bas
```

### concat

```sh
cargo run -- concat
```
