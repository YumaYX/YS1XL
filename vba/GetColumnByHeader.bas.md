# GetColumnByHeader

- 目的: 指定されたワークシートの指定された行から、与えられた見出し名に対応する列番号を検索する。

# 入力（日本語）

| 引数名 | 型 | 説明 |
|--------|----|------|
| ws | Worksheet | 列を検索する対象のワークシート。 |
| header | String | 検索したい見出しのテキスト。 |
| rowNum | Long | 見出しが配置されている行番号（デフォルトは1）。 |

# 出力（日本語）

- 型: Long
- 内容: 見出しが見つかった場合はその列番号、見つからなかった場合は0を返す。
