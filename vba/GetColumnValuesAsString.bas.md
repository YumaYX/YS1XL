# GetColumnValuesAsString

- 目的: 指定されたワークシートの指定列の全ての値を、区切り文字でつないだ単一の文字列として返します。

# 入力

| 引数名 | 型 | 説明 |
|--------|----|------|
| ws | Worksheet | 値を取得する対象のワークシート。 |
| colNum | Long | 値を取得する列番号（省略時：1列目）。 |
| delimiter | String | 値と値の間に入れる区切り文字（省略時：改行コード）。 |

# 出力

- 型: String
- 内容: 指定された列の1行目から最終行までの全ての値が、指定された区切り文字で結合された文字列。
