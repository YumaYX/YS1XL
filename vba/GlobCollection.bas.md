# GlobCollection

- Purpose: Returns a collection containing the full paths of all files found within a specified directory that match a given pattern.

# Input

| Argument Name | Type | Description |
|---------------|------|-------------|
| folderPath    | String | The directory path where the search should begin. |
| pattern       | String | The file name pattern to match (e.g., "*.txt"). |

# Output

- Type: Collection
- Content: A collection containing the full file paths (Strings) of all files that matched the specified pattern.
