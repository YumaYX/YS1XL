# 関数ドキュメント: ReadUtf8Text

## 概要
指定されたファイルパスからファイルの内容を読み込み、UTF-8エンコーディングの文字列として返します。この関数は、バイナリデータではなく、テキストファイル（特にUTF-8形式）の読み取りに特化しています。

## 構文
```vba
ReadUtf8Text(filePath As String) As String
```

## パラメータ
| 名前 | 型 | 説明 |
| :--- | :--- | :--- |
| `filePath` | `String` | 読み込むファイルの完全なパス。 |

## 戻り値
| 型 | 説明 |
| :--- | :--- |
| `String` | ファイルの内容全体を格納した文字列。エンコーディングはUTF-8として処理されます。 |

## 詳細
*   **処理内容:** `ADODB.Stream`オブジェクトを使用してファイルストリームを操作します。ストリームの`Charset`を明示的に`"UTF-8"`に設定することで、文字化けを防ぎ、正確なUTF-8テキストとして読み込みます。
*   **前提条件:** この関数を実行するには、VBA環境で`Microsoft ActiveX Data Objects Library`への参照設定（または実行時オブジェクトの生成）が必要です。
*   **エラー処理:** ファイルが存在しない場合やアクセス権がない場合、実行時エラーが発生する可能性があります。
