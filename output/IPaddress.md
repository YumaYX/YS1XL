# ネットワークアドレスユーティリティ関数群

本モジュールは、IPv4アドレス、サブネットマスク、およびCIDR表記に関する検証および変換を行う一連の関数群です。ネットワークアドレスの妥当性チェックや、表記形式間の相互変換をサポートします。

## 関数一覧

### `IsValidIPAddress(ByVal ip As String) As Boolean`
指定された文字列が有効なIPv4アドレス形式であるかを検証します。
*   **引数:** `ip` (String) - 検証対象のIPアドレス文字列。
*   **戻り値:** Boolean - 有効なIPアドレスであれば `True`、そうでなければ `False`。
*   **検証ロジック:**
    1.  ドット（`.`）で区切られた4つの部分を持つかを確認します。
    2.  各部分が数値であり、範囲が0から255の間であるかを確認します。
    3.  先頭ゼロ（例: "01"）による不正な表現を検出します。

### `IsValidSubnetMask(ByVal mask As String) As Boolean`
指定された文字列が有効なサブネットマスクであるかを検証します。
*   **引数:** `mask` (String) - 検証対象のサブネットマスク文字列。
*   **戻り値:** Boolean - 有効なサブネットマスクであれば `True`、そうでなければ `False`。
*   **検証ロジック:**
    1.  構造（4つのオクテット）と数値範囲（0-255）をチェックします。
    2.  バイナリ表現が「1が連続し、その後0が連続する」というマスクの定義パターン（`01`のパターンが存在しない）を満たしているかを厳密にチェックします。
    3.  全ビットが1（255.255.255.255）または全ビットが0（0.0.0.0）のマスクは無効として扱います。

### `IsValidNetworkAddress(ByVal ip As String, ByVal mask As String) As Boolean`
与えられたIPアドレスが、指定されたサブネットマスクによって定義されるネットワークアドレスに属するかを検証します。
*   **引数:**
    *   `ip` (String) - 検証対象のIPアドレス。
    *   `mask` (String) - ネットワークを定義するサブネットマスク。
*   **戻り値:** Boolean - IPアドレスがマスクによって定義されるネットワークアドレス範囲内であれば `True`、そうでなければ `False`。
*   **ロジック:** IPアドレスとマスクをビット単位AND演算（`IP AND MASK`）を行い、その結果がネットワークアドレスとして妥当であるかを検証します。

### `CIDR2Mask(cidr As Integer) As String`
CIDR表記（クラスレスドメインルーティング）のプレフィックス長を、ドット区切りのサブネットマスク文字列に変換します。
*   **引数:** `cidr` (Integer) - CIDRプレフィックス長（例: 24）。
*   **戻り値:** String - 対応するサブネットマスク（例: "255.255.255.0"）。
*   **ロジック:** CIDR値に基づいて、各オクテットのビットパターンを計算し、ドット区切りで結合します。

### `Mask2CIDR(mask As String) As Integer`
ドット区切りのサブネットマスク文字列を、対応するCIDRプレフィックス長（整数）に変換します。
*   **引数:** `mask` (String) - 入力されたサブネットマスク文字列。
*   **戻り値:** Integer - 対応するCIDRプレフィックス長（例: 24）。
*   **ロジック:** マスクの各オクテットをバイナリに分解し、セットされている'1'の総数を数えることでCIDR値を算出します。
