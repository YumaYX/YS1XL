# 関数名: GetTimestamp
## 概要
現在の日時を特定の文字列形式（`yyyy-mm-dd-HH-MM-ss`）で取得し、返します。システム時刻をログ記録やデータ処理のタイムスタンプとして使用する際に適しています。

## 構文
`GetTimestamp()`

## パラメータ
| パラメータ名 | 型 | 説明 |
| :--- | :--- | :--- |
| なし | - | 引数は必要ありません。 |

## 戻り値
| 型 | 説明 |
| :--- | :--- |
| `String` | 現在のシステム時刻を「年-月-日-時-分-秒」の形式で表した文字列。例: "2023-10-27-14-30-55" |

## 使用例
```vb
Dim timestamp As String
timestamp = GetTimestamp()
' timestamp には現在の日時が格納される
MsgBox "現在のタイムスタンプ: " & timestamp
```
