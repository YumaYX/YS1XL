# プログラム概要とドキュメント生成

## 概要
本プログラムは、指定されたCSVファイル（またはExcelファイル）からデータを読み込み、特定のキー列（例：「id」）に含まれる一意な値と、その値がデータセット内で最初に出現した行番号を対応付けた辞書（Dictionary）を生成します。

主に、データセット内のキー値の重複チェックや、特定のキー値がどの行に存在するかを高速で検索するために使用されます。

---

## 関数詳細

### 1. StoreDataLocation(data As Variant, keyIndex As Long)
**機能:** 2次元配列データと指定された列インデックスを受け取り、その列の各要素をキー、対応する行番号を値とする辞書オブジェクトを生成します。
**引数:**
*   `data` (Variant): 処理対象の2次元配列データ（ワークシートの範囲データなど）。
*   `keyIndex` (Long): キーとして使用する列のインデックス番号（1から始まる）。
**戻り値:**
*   `Object` (Scripting.Dictionary): キーがデータ列のユニークな値、値がその値が最初に見つかった行番号（1から始まる）である辞書オブジェクト。

### 2. SearchDataLocation(Optional csvFilePath As String = "sample.csv", Optional targetKey As String = "id")
**機能:** 指定されたファイルパスからデータを読み込み、ヘッダー名に基づいてターゲットキー列を特定し、その列のデータを用いてデータロケーション辞書を生成します。
**引数:**
*   `csvFilePath` (Optional String): 処理対象のCSVファイルパス。デフォルトは "sample.csv"。
*   `targetKey` (Optional String): 辞書を生成する際に使用するキー列のヘッダー名（例: "id"）。デフォルトは "id"。
**処理フロー:**
1.  指定されたファイルを開き、最初のシートの全使用範囲を配列として読み込みます。
2.  1行目（ヘッダー行）を解析し、ヘッダー名と列インデックスをマッピングした辞書を作成します。
3.  `targetKey`に対応する列インデックスを取得します。
4.  取得したデータ配列と列インデックスを`StoreDataLocation`関数に渡し、最終的なロケーション辞書を生成します。
5.  ワークブックを閉じます。
**戻り値:**
*   `Object` (Scripting.Dictionary): キーがデータ列のユニークな値、値がその値が最初に見つかった行番号である辞書オブジェクト。
