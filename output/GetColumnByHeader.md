### 関数名: GetColumnByHeader
### 説明:
指定されたワークシートの指定行において、特定のヘッダー名（見出し）を持つセルが配置されている列番号を検索し、返します。

### 構文:
```vba
Function GetColumnByHeader(ws As Worksheet, header As String, Optional rowNum As Long = 1) As Long
```

### パラメータ:
| パラメータ名 | 型 | 説明 |
| :--- | :--- | :--- |
| `ws` | `Worksheet` | 検索対象とするワークシートオブジェクト。 |
| `header` | `String` | 検索したい見出し名（ヘッダー文字列）。 |
| `rowNum` | `Long` | 見出しが配置されている行番号。省略した場合、デフォルトで1行目（`1`）が使用されます。 |

### 戻り値:
*   **`Long`**: 見出しが発見された場合の列番号（1以上の整数）。
*   **`Long`**: 見出しが見つからなかった場合は `0` を返します。

### 動作概要:
1.  指定された`ws`の`rowNum`行目において、左端から右へ順にセルを走査します。
2.  各セルの値が`header`引数と一致するかどうかをチェックします。
3.  一致した場合、そのセルの列番号を即座に返して処理を終了します。
4.  最後まで検索しても一致するものがない場合、`0`を返します。
