# プログラムの説明とドキュメント

## 概要
本プログラムは、Outlookの指定されたメールフォルダ内を検索し、添付ファイルの内容をローカルディスクに保存するVBAサブルーチンです。添付ファイルは、ファイル名に指定されたキーワードが含まれている場合にのみ処理され、かつ既に保存先フォルダに同じファイルが存在しない場合に限り、重複保存を防ぎながら保存されます。

## サブルーチン名
`SaveAttachments`

## 構文
```vba
Public Sub SaveAttachments( _
    ByVal mailFolderPath As String, _
    ByVal targetFolderPath As String, _
    ByVal keyword As String)
```

## 引数 (Parameters)

| 引数名 | 型 | 説明 |
| :--- | :--- | :--- |
| `mailFolderPath` | `String` | 処理対象とするOutlookメールフォルダのパス（ルートフォルダからの相対パス）。例: "アーカイブ" |
| `targetFolderPath` | `String` | 添付ファイルを保存するローカルディレクトリのパス。 |
| `keyword` | `String` | 添付ファイル名に含まれる必要があるキーワード。大文字・小文字を区別しません。 |

## 処理の流れ (Logic Flow)

1.  **Outlook接続の確立:**
    *   Outlookアプリケーションに接続を試みます（既存インスタンスを優先）。
    *   接続に失敗した場合、新しいOutlookインスタンスを作成します。
    *   MAPI名前空間を取得し、デフォルトフォルダの親（ルート）を取得します。
2.  **フォルダのナビゲーション:**
    *   `mailFolderPath`を受け取り、パスの区切り文字（`\`）で分割します。
    *   分割されたパス要素を順にたどり、指定されたメールフォルダに到達します。
3.  **メールアイテムのループ処理:**
    *   指定されたフォルダ内のすべてのアイテムをループ処理します。
    *   アイテムがメール（`MailItem`）であるかを確認します。
4.  **添付ファイルの処理:**
    *   メールに添付ファイルが存在する場合、添付ファイルを一つずつループ処理します。
    *   **キーワードチェック:** 添付ファイル名（`olAttachment.FileName`）を小文字に変換し、指定された`keyword`が含まれているか（`InStr`）を確認します。
    *   **保存処理:**
        *   キーワードが一致した場合、保存先のフルパスを構築します。
        *   `Dir()`関数を使用して、既にそのパスにファイルが存在するかを確認します。
        *   ファイルが存在しない場合のみ、`olAttachment.SaveAsFile`メソッドを使用してローカルディスクにファイルを保存します。

## 使用例
```vba
' SaveAttachments("アーカイブ", "C:\Temp", "キーワード")
' フォルダパス: "アーカイブ"
' 保存先パス: "C:\Temp"
' 必須キーワード: "キーワード"
```
